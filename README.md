看到Awesome ABC​的作品，我试用了一下很好用，但有一些便捷性问题，因为这个签名每次重启客户端会自己改回来，所以频繁使用的话需要频繁重新输入指令和内容，有一些麻烦。从贴吧npc（https://tieba.baidu.com/p/9035884910?see_lz=1）这里看来了一些方法，npc的缺点可能是封装包太大，75Mb，于是我写了个由Powershell、txt、bat文件组合的绿色版本，zip文件2kb。 AI赋能，代码标注可读，清晰透明。看到还有拿这种软件卖钱的，便把这个简单的工具拿出来给大家分享。

使用方法：

1、下载全部三个文件。

2、将自己想要的签名内容写入signature.txt文件中。

3、在英雄联盟客户端已登录的情况下运行run_lol_signature_updater.bat文件，当询问管理员权限时同意。

4、修改完成。



适配性理应没太大问题，解压后保证三个文件在同一目录下，bat文件会需要以管理员权限启动，cd到当前文件夹，运行ps1文件来获得英雄联盟客户端的token、端口，然后将txt文件中的内容PUT到客户端。
